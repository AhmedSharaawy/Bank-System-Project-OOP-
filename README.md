# 🏦 Bank System Console Based Banking Management (C++ OOP)

## Overview

A console based **Bank Management System** built using **C++ (OOP)** to simulate a real-world banking environment.

The system manages clients, financial transactions, users, and currencies while ensuring secure access control and persistent data storage.

---

## 🔐 Authentication and Permissions

- The system starts with a secure login process
- User credentials are validated before access is granted
- Each user has assigned permissions
- Every operation is checked against user permissions before execution
- Ensures controlled and secure system access

---

## 👤 Client Management

- Add new clients with unique account numbers
- Update client information
- Delete clients with confirmation
- Search for specific clients
- List all client accounts
- Calculate total balances
- All changes are saved immediately to maintain data integrity

---

## 💰 Transactions

- Deposit money into accounts
- Withdraw money with balance validation
- Transfer funds between accounts
- All transfers are recorded in a dedicated log file
- Ensures accurate transaction tracking

---

## 👥 User Management

- Clear separation between bank staff (users) and clients
- Full management of system users
- Assign and update user permissions
- Secure control over system access

---

## 💱 Currency System

- List available currencies
- Search for a specific currency
- Update exchange rates
- Perform currency conversions using reference values
- Ensures accurate and dynamic calculations

---

## 🗂 Data Persistence

- All data is stored in external files
- Data is loaded automatically at system startup
- Maintains continuity between sessions
- Ensures reliable and consistent storage

---

## 🎯 System Design Concept

The system is structured around:

- Organized menu driven navigation
- Clear role based access control
- Reliable transaction logging
- Persistent file based data storage

This project simulates a real banking system environment within a console application while applying core Object-Oriented Programming principles.
